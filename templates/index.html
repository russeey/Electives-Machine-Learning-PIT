About This System

Developers: Dapar, Eldrich James & Ravidas, Russel Ky

Subject Instructor: Engr. Jodie Rey Fernandez

Overview: This system predicts earthquakes in the Philippines by analyzing historical seismic data and applying machine learning models to estimate the probability of occurrence.

How It Works:

  • Historical earthquake data is collected and processed.
  • Features are extracted and normalized.
  • A machine learning model predicts expected earthquake counts.
  • The system converts predictions into probabilities and intensity categories.
  • Users input a month and year for forecast.
